## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd talent-tree-project
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
